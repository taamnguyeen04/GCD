# Ingest / Query / Lint Loop

## Definition
A three-operation cycle for knowledge compounding: ingest new evidence, query compiled knowledge, and lint the structure for drift and gaps.

## Why It Matters
It turns ad-hoc note accumulation into a maintainable system with explicit upkeep.

## Evidence
The source defines each operation and recommends logging and indexing as mandatory support files.
